 Here are the test classes created for different components of the project:

OsifJavaCgCandidateApplicationTest
Tests the main application context.
CandidateConfigTest
Ensures that the application context loads successfully.
CandidateControllerTest
Tests the REST controller for adding, retrieving, and listing candidates.
CandidateModelTest
Verifies the equality, inequality, hash code, and string representation of the Candidate model.
CandidateServiceTest
Tests the service layer for adding, retrieving, and listing candidates, including interactions with the repository.
Each test class is structured to cover various scenarios and uses mock objects where necessary to isolate the components being tested. If you need the test files or have any specific requests regarding the tests, let me know!
